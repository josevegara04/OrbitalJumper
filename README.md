# OrbitalJumper

Orbital Escape es un juego hipercasual desarrollado en Unity donde el jugador debe navegar entre planetas utilizando su gravedad para impulsarse de una órbita a otra.

El objetivo es sobrevivir el mayor tiempo posible saltando entre planetas sin perderse en el espacio. Cada salto requiere precisión y buen timing, ya que la posición y distancia entre planetas cambia constantemente.
